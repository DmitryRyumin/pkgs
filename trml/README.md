# Работа с Shell

[![PyPI version](https://badge.fury.io/py/trml.svg)](https://badge.fury.io/py/trml)

## Установка

```shell script
pip install trml
```

## Обновление

```shell script
pip install --upgrade trml
```

## Автоматические скрипты

| Файлы | Описание|
| ----- | ------- |
| trml_add_line | Добавление линии во весь экран |
| trml_clear | Очистка консоли |
